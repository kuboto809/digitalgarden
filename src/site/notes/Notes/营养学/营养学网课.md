少于 2%的成分可以不按顺序写。

<font color=#07f2d7>糖类</font>：白砂糖、葡萄糖、麦芽糖（饸糖）、果葡糖浆、琴麦芽糖浆、糊精、淀粉。
意义：这些糖类不但没有什么营养价值，而且几乎都是高 GI 食物（应当避免）。

<font color=#07f2d7>油脂</font>：植物油、<font color="#e5b9b7">精炼植物油、氢化植物油（奶油、奶茶）、植物起酥油（加在面包中，让面包压缩性更好、更蓬松）</font>、植物黄油（奶油）、棕榈油、椰子油等。
意义：添加油脂后，食品的脂肪和能量大增。氢化植物油、植物起酥油、植物黄油等含有较多反式脂肪酸，对血脂有不良影响。棕榈油、椰子油、动物油等含有较多饱和脂肪酸，也对血脂不利。

<font color=#07f2d7>钠</font>：包括食盐（氯化钠）、苯甲酸钠磷酸钠、糖酸氢钠、谷氨酸钠、亚硝酸盐、异维生素 C 钠等。
意义：食品中不同来源的钠盐作用各不同，有的调味、有的防腐，有的提高稳定性，有的上色，但有一点是共同的，它们都会影响血压，对防治高血压有害。

<font color=#07f2d7>营养添加物</font>：蛋白质、卵磷脂、各种维生素和矿物质等。例如鸡蛋、奶粉、乳清粉、维生素 C、B 族维生素、维生素 E、胡萝卜素、钙、铁、锌等。
意义：能提高食品的营养品质。

<font color=#07f2d7>胶质添加物</font>：卡拉胶、黄原胶、瓜尔豆胶、刺槐豆胶、海藻胶（海藻酸钠）、琼脂、魔芋胶、食用明胶等。（明胶是动物胶，其他都是植物胶）
作用：加胶可<span style="background:rgba(240, 107, 5, 0.2)">使饮料、或奶制品浓稠</span>，果冻成型、<span style="background:rgba(240, 107, 5, 0.2)">火腿肠赋于弹性和光滑度</span>等。

分类：一类是<span style="background:rgba(240, 107, 5, 0.2)">植物来源</span>的，如卡拉胶、黄原胶、瓜尔豆胶、刺槐豆胶、海藻胶、琼脂、魔芋胶等。其性质<font color="#d7e3bc">是膳食纤维</font><font color="#d7e3bc">（吃多了会抑制阳离子的吸收）</font>。另一类是<span style="background:rgba(240, 107, 5, 0.2)">动物来源</span>的，如食用明胶等。是由动物皮屑、软骨、韧带、肌膜等含的的胶原蛋白。

# 食品添加剂
## （一）概述
![Pasted image 20230708223804.png](/img/user/Notes/%E8%90%A5%E5%85%BB%E5%AD%A6/attachments/Pasted%20image%2020230708223804.png)
食品强化剂：在食品中添加的营养成分，如在奶粉中添加对成长发育有益的成分（如铁、锌）；在食盐中加碘等。
我国法律中的食品添加剂包含食品强化剂。
<font color=#07f2d7>（二）食品添加剂的作用</font> 
1. 有益于食品的保藏，防止食品腐败变质（特别是预包装食品）（水分易导致腐坏）；
2. 改善食品的感官性状（<span style="background:rgba(240, 107, 5, 0.2)">色素</span>、<span style="background:rgba(240, 107, 5, 0.2)">增稠剂</span>）；
3. 保持或提高食品的营养价值；
4. 增加食品的品种和方便性 （液态-> 固态）；
5. 有利于食品的加工操作，适应生产的机械化和自动化；
6. 满足不同人群的特殊营养需要。
<font color=#07f2d7>（三）食品添加剂的分类</font>
1. 按用途分类：防腐剂、抗氧化剂、着色剂、护色剂、票白剂、增味剂、凝固剂、膨松剂、增稠剂、消泡剂、甜味剂、乳化剂、面粉处理剂、水分保持剂、抗结剂、曾味剂、酸度调节剂、酶制剂、被膜剂、香料、<span style="background:rgba(240, 107, 5, 0.2)">营养强化剂</span>、加工助剂及其它等类。（23 类）
2. 按来源分类：天然食品添加剂、化学合成食品添加剂
<font color=#07f2d7>（四）食品添加剂对健康的影响</font>
化学合成的食品添加剂具有一定的毒性，但<font color="#d99694">毒副作用是在达到一定浓度和剂量时才会产生</font>。在规定的浓度和剂量范围内是安全的。
长期摄入食品添加剂可能带来的潜在危害。目前我国已批准使用的共 1000 多种。超过 95%的加工食品都使用了食品添加剂。人均每天摄取的添加剂大约是 10 克，每天至少吃进百余种食品添加剂。<span style="background:rgba(240, 107, 5, 0.2)">(中国人均每天摄取的食盐在 10 克 -12 克)</span>。
天然食物中的食品添加剂也对健康稍有影响。
## 二、常见的食品添加剂
### (一) 防腐剂
防腐剂是最早使用的食品添加剂中的一种，它用于防止食品因微生物引起的变质，提高食品保存性能，延长食品保质期等等。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"><b>1. 苯甲酸及其钠盐</b></span></div></div>苯甲酸又名安息香酸，在酸性条件下对多种微生物（酵母、霉菌、细菌）有明显的<span style="background:rgba(240, 107, 5, 0.2)">抑菌作用</span>。由于苯甲酸溶解度低，实际生产中大多使用其钠盐。苯甲酸类通常使用在<span style="background:rgba(240, 107, 5, 0.2)">果酱类、酱菜类</span>、和一些饮料等食品中。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"><b>2.山梨酸及其盐类</b></span></div></div>
山梨酸又名花秋酸，为酸性防腐剂具有较高的<span style="background:rgba(240, 107, 5, 0.2)">抗菌</span>性能，对细菌、霉菌、酵母菌均有抑制作用，<font color="#d99694">而毒性仅为苯甲酸的 1/4（但成本更高）</font>。常用于一些<span style="background:rgba(240, 107, 5, 0.2)">饮料、果脯、肉制品</span>等食品中。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"><b>3. 乳酸链球菌素</b></span></div></div>
是乳酸链球菌产生的一种多肽物质，由 34 个氨基酸残基组成，是一种高效、无毒、安全、<span style="background:rgba(240, 107, 5, 0.2)">无副作用</span>的<span style="background:rgba(240, 107, 5, 0.2)">天然</span>食品防腐剂。用于<span style="background:rgba(240, 107, 5, 0.2)">乳制品、肉制品、植物蛋白食品</span>的防腐保鲜。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"><b>4. 其它的防腐剂</b></span></div></div>
对羟基甲酸酯类、丙酸钠、丙酸钙、双乙酸钠、乳酸钠。

#### 本产品不含防腐剂”你相信吗？
<font color=#07f2d7>（一）不需要填加防腐剂的食品</font>
1. 水分含量低的食品：方便面、饼干
2. 高糖或高盐食品：蜜饯、腌菜<font color="#d99694">（渗透压高，细胞形态难保持）</font>
3. 杀菌密封食品：罐头、真空包装食品
4. 本身有杀菌作用的食品：酸奶<font color="#d99694">（里面的乳酸菌已有抑菌作用）</font>、蜂蜜
<font color=#07f2d7>（二）不充许添加防腐剂的食品</font>
奶粉、食用植物油、瓶（桶）装饮用水、糖果（硬糖、酥糖、充气糖）等。

### （二）抗氧化剂
主要功能是防止或减慢食品发生氧化作用，它可以有效防止食品中脂类物质白的氧化，避免发生酸败。
>[!Abstract]+ 氧化
>氧在我们体内发生化学反应时会产生过氧化物，过氧化物生成自由基，自由基和人体内的负离子发生反应，缺乏负离子导致细胞变异，比如癌细胞。变异了的细胞与其他细胞争夺负离子，进而导致更多细胞变异。所以自由基数量越多，人体健康状况越差。
>维 C（绿叶、黄绿色蔬果）、维 E（植物种子）、维 A（动物肝脏、血制品；绿叶、黄绿叶蔬菜中含类胡萝卜素）、微量元素硒（大蒜）都可抗氧化。 

<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"><b>天然合成的抗氧化剂</b></span></div></div>
<span style="background:rgba(240, 107, 5, 0.2)">1. 天然 VE</span>：存在于植物油脂中，状态比较稳定、抗氧化性较好，使用安全。主要用于婴儿食品、奶粉。
<span style="background:rgba(240, 107, 5, 0.2)">2. 茶多酚类</span>：存在于茶叶中提取的抗氧化物质。抗氧化能力比 VE 强。
3. VC （抗坏血酸）、红辣椒提取物、香辛料提取物、糖醇类。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"><b>人工合成的抗氧化剂</b></span></div></div>
二丁基羟基甲苯 (BHT)，
叔丁基对羟基茴香醚（BHA) ，
没食子酸丙酯 (PG)、
叔丁基对苯二酚 (TBHQ)。
<font color=#07f2d7>食品抗氧化剂应具备以下条件：</font>
①具有优良的抗氧化效果；
②本身及分解产物都无毒无害；
③稳定性好，与食品可以共存；对食品的感官性质 (包括色、香、味等) 没有影响；
④使用方便，价格便宜。

### (三）甜味剂
是指赋予食品以甜味的食品添加剂，甜度是蔗糖的几十倍到几百倍。可以分为糖类非糖类，营养、非营养，人工合成、天然甜味剂等。
吃甜食时体内会释出多巴胺、内啡肽，让我们快乐。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"></span><b>1. 糖精</b></div></div>
又称为糖精钠，是<span style="background:rgba(240, 107, 5, 0.2)">从煤焦油中提炼</span>出来的，是最古老的化学合成甜味剂，甜度是蔗糖的 400-700 倍，用途广泛。<span style="background:rgba(240, 107, 5, 0.2)">其浓度稍大时会有苦味</span>，另外，<span style="background:rgba(240, 107, 5, 0.2)">有致癌性</span>，在许多国家的一些食品中禁用，我国不充允许在婴儿食品中使用。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"></span><b>2. 甜蜜素</b></div></div>
化学名称是环己基磺酸钠，其甜度是蔗糖的 30 倍，口味极似蔗糖；无余苦味，而且耐候性好。由于 20 世纪 70 年代曾发现甜密素对动物有致癌的现象，被禁用过，后来由于大量试验表明它并无致畸、致癌作用等，许多国家重又许可使用。我国于 1987 年开始应用甜蜜素，目前仍在应用。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"></span><b>3. 阿斯巴甜</b></div></div>
化学名称为天门冬酰苯丙氨酸甲酯，甜味纯正无任何后味，是迄今开发成功的甜味剂中甜味最接近蔗糖的（蔗糖为 1，它为 0.6），在食品中使用广泛。阿斯巴甜的安全性在使用早期引起了广泛的争议，但<font color="#d99694">自前还没发现阿斯巴甜而对人体产生危害</font>或致癌的案例现在认为它在规定的范围内是安全的。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"></span><b>4. 安赛蜜</b></div></div>
化学名称是乙酰磺胺酸钾，是目前世界上第四代合成甜味剂。甜味纯正而强烈，优于蔗糖甜味，<font color="#d99694">持续时间长</font>，与阿斯巴甜 1：1 合用有明显增效作用。它的<font color="#d99694">安全性较高</font>，联合国 FAOWHO 联合食品添加剂专家委员会同意安赛蜜用作<font color="#d99694"> A 级食品</font>添加剂（添加后仍为绿色食品），并推荐日均摄入量（ADI）为 0~15mg/kg。它的生产工艺简单、价格便宜、性能优于阿斯巴甜，被认为是最有前途的甜味剂之一。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"></span><b>5. 低聚果糖</b></div></div>
低聚果糖是<font color="#d99694">天然的</font>营养型甜味剂。它具有某些独特的生理功能<font color="#d99694">促进体内双歧杆菌的增殖</font><font color="#d99694">（益生菌）</font>，提高人体免疫力；低能量，很难被人体消化吸收，适用于糖尿病肥胖病、高血压患者 ; 具有防隅齿功能（抑制口腔中的致病菌）；属于<font color="#d99694">水溶性膳食纤维</font>。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"></span><b>6. 木糖醇</b></div></div>
是<font color="#d99694">天然的</font>功能性甜味剂，其主要的生理功能类似低聚糖。木糖醇存在于多种水果和蔬菜中，目前工业上主要从玉米芯、甘蔗渣等植物中提取木糖醇。其甜度与蔗糖相等。从 70 年代未到 80 年代初许多国家陆续将其作为营养性甜味剂列入添加剂标准。我国目前有含木糖醇的<font color="#d99694">口香糖、奶糖、糕点、饮料</font>和营养液上市。木糖醇不会干扰血糖，所以可供糖尿病患者使用。

#### “无糖食品真的健康吗？
按照国际通用的概念，无糖食品<span style="background:rgba(240, 107, 5, 0.2)">不能含有蔗糖和来自于淀粉水解物的糖</span>，包括<font color="#d99694">葡萄糖、麦芽糖、果糖</font>等。但是，<span style="background:rgba(240, 107, 5, 0.2)">它必须含有相当于糖的替代物</span>，一般采用糖醇或低聚糖等不升高血糖的<span style="background:rgba(240, 107, 5, 0.2)">甜味剂</span>品种。
事实上现在市场上号称的“无糖食品”中可能含有<span style="background:rgba(240, 107, 5, 0.2)">淀粉水解物</span>作为甜味来源，即淀粉糖浆、果葡糖浆、麦芽糖之类。这些糖浆升高血糖、变成能量的效率，未必比蔗糖慢。
#### 木糖醇并不是多多益善
现在很多“无糖”食品以木糖醇作为卖点，因为木糖醇作为一种功能性甜味剂，具有热量低，代谢速度快，不会引起血糖值升高的特点。但是，木糖醇也并非多多益善。木糖醇<font color="#d99694">不被人体内的胃酶分解，直接进入肠道，易造成渗透性腹泻</font>。在欧美国家，含有木糖醇的食品都会在标签上注明“过量摄取可能会导致腹泻”，但国内市场销售的食品却鲜见这种标识。
#### 警惕 “无糖”食品中的合成甜味剂
<font color="#d99694">合成甜味剂</font>是标明或宣传专“低糖”“低热量”、“减肥”的甜味食“无糖”品和饮料当中最常见的甜味来源，既便宜又稳定，<font color="#d99694">不含热量</font>，也不会<font color="#d99694">引起隅齿</font>。但是，专家认为<font color="#d99694">儿童不适宜</font>吃含有化学合成甜味剂的食品。因为化学合成的甜味剂食用后不易分解，<font color="#d99694">易对肝脏造成损害</font>。而儿童的免疫系统尚未发育成熟，肝脏的解毒能力较弱。还有部分健康专家担心儿童食用含有合成甜味剂的食物对<font color="#d99694">心理、行为发育</font>可能造成潜在不良影响（依赖性）。孕妇最好也避开这类食品。

### （四）着色剂
食用着色剂又称食用色素，是使食品着色，从而改善食品色调和色泽的可食用物质。分为人工合成色素和天然色素两大类。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"></span><b>1. 人工合成色素</b></div></div>
人工合成色素是从<font color="#d99694">石油或煤焦油</font>中提炼出来，具有提炼方法简单、<font color="#d99694">色泽鲜艳</font>、<font color="#d99694">着色力强</font>、性质稳定和价格便宜等优点，被食品企业广泛使用。
但是，人工合成色素在经化学方法提取与合成过程中，<font color="#d99694">可能混入砷、汞、苯胺等杂质</font>；合成色素的摄入过多会对人的身体产生不良影响。
目前，我国批准在食品上使用的合成色素有苋菜红、胭脂红、新红、柠檬黄、日落黄、靛蓝、亮蓝、赤藓红、诱惑红、二氧化钛等。
它们分别被用于蜜饯、糕点、糖果、果冻、果汁、软饮料、罐头等食品表面的上彩。<font color="#d99694">婴幼儿、主食类、肉制品、鱼制品、调味品、水果、乳制品、饼干、炒货等食品<font color="#d99694">禁止使用</font></font>人工合成色素。
<div class="emphasis-block yellow"><div class="side-peice">&nbsp</div><div class="text-block"><span class="text-inner"></span><b>2. 天然色素</b></div></div>
是从植物、微生物、动物可食性部分用物理方法提取精制而成的，安全性高。有些天然色素，<font color="#d99694">对人体是一种有益补充</font>，如番茄红素。我国批准允许使用的食用天然色素包括天然胡萝卜素、甜菜红、姜黄、辣椒红、子黄、焦糖色、胭脂虫红、红花苷、红曲红等 50 多种。但由于其来源较少不易提炼、<font color="#d99694">价格偏高、着色力差</font>，食品生产企业不愿使用。<font color="#d99694">有些化妆品中就添加了天然色素</font>（因此声称可少量食用）。

## 三、正确解读食品添加剂
<font color=#07f2d7>（一）少吃加工食品</font>：最好的办法就是将食品原材料买回家，自己动手做。把新鲜的烹饪原料买回家做的话，每天摄入的食品添加剂就非常少了。但是，如果使用半成品进行烹饪，那么吃进去的食品添加剂会更多。
<font color=#07f2d7>（二）购物时看清原料说明</font>：购物时要养成看配料表的习惯，从配料表中我们可以清楚地看到食品中的添加剂，应尽量选一些添加天然添加剂的食品。
<font color=#07f2d7>（三）选择诚信企业</font>：在购买食品时应选一些有诚信的企业。<font color="#d99694">有些小企业在配料表上做文章，明明加了三种色素，只标示“食用色素”的字样，误导消费者。</font>
<font color=#07f2d7>（四）不要追求特别漂亮的颜色和浓烈的口感</font>：<font color="#d99694">越是保质期长的</font>，口感出色的，颜色漂亮的，<font color="#d99694">打开包装就能吃的食品</font>，在配料中就越离不开食品添加剂。

## 四、实例分析
>[!Abstract] 酸度调节剂
>通过调节食品体系中的酸碱性可以保持食品的最佳形态和韧度（果胶的凝胶、干酪的凝固尤其如此），同时降低体系的 pH 值，达到抑菌目的，多数细菌耐受 pH 为 6.5—7.5，少数耐受 pH 为 3—4 的范围（如酵母菌、霉菌）。
### （一）酱油
酿造酱油：防腐剂（苯甲酸钠、山梨酸钾）通常也是价格差别所在。
配制酱油：焦糖色素、增味剂 (谷氨酸钠、肌苷酸二钠、鸟苷酸二钠)、乳酸（酸度调节剂）、香味剂 (里面可包含 N 种化合物)、防腐剂 (山梨酸钾等)。
### （二）火腿肠
含有水分保持剂、增稠剂（卡拉胶等）、食用香料增味剂（谷氨酸钠等）、山梨酸钾、亚硝酸钠（防腐剂）、红曲米、D-异抗坏血酸钠（抗氧化剂）、酸度调节剂（乳酸）等食品添加剂。

## 五、国家对食品添加剂的管理措施
<font color=#07f2d7>（一）限制添加剂的种类。</font>
食品中可以添加的添加剂必须是国家允许使用的，否则是违法的。
<font color=#07f2d7>（二）限制使用范围。</font>
某种添加剂只能使用于国家允许的食品种类中，像婴儿食品中是不允许使用绝大多数的食品添加剂的。
<font color=#07f2d7>（三）限制添加的数量。</font>
食品中的添加剂必须符合国家所规定的最大允许残留量。

# 体检报告解读
## 常见体检项目
常见的体检项目包括：<font color="#d99694">物理检查</font>（身高、体重、BMI、血压）、<font color="#d99694">内科</font>（触诊、问诊）、<font color="#d99694">外科</font>（观察器官）常规检查，<font color="#d99694">B 超</font>（肝、胆、脾、胰、双肾），<font color="#d99694">心电图</font>，<font color="#d99694">胸片</font>（X 片、CT），<font color="#d99694">血生化检查</font> (血尿常规、肝功能、血脂四项、尿素氮、血肌酐、尿酸、血糖、同型半胱氨酸癌症指标等）。职业健康体检除以上项目外，粉尘作业职工加做高干伏胸片、肺功能检查，噪声作业职工加纯音测听检查。

### 物理检查
<font color=#07f2d7>1. 血压</font>
《2014 年中国高血压防治指南》血压从 <span style="background:rgba(240, 107, 5, 0.2)">110/75mmHg</span> 起，人群血压升高水平与心血管疾病显正相关。<span style="background:rgba(240, 107, 5, 0.2)">正常高值血压 120~139/80~89</span> 应改善生活方式，预防高血压的发生。血压 140/90，同一时间三次及以上可以诊断并进行治疗。
收缩压正常值：90 - 120 mmHg
舒张压正常值：60 - 90 mmHg
压差：大于等于 30 mmHg
早上血压最高，早中晚测量时，只要有一次高于正常值，就算是不正常，不要取早中晚平均。但同一时间测量时可取平均，如早上测两次取平均。
高血压患者应<span style="background:rgba(240, 107, 5, 0.2)">注意钠的摄入量</span>，同时可摄取<span style="background:rgba(240, 107, 5, 0.2)">丰富钙的食物</span>，钙帮助降低血压。

### 血生化检查
#### 肝功能检查项目
<font color=#07f2d7>1. 谷丙转氨酶（ALT）</font>
谷丙转氨酶正常< 40 u/L ，增高见于<font color="#d99694">肝胆疾病</font>、传染性肝炎、中毒性肝炎、肝癌、肝硬化、脂肪肝、梗阻性黄疸、胆管炎、胆囊炎等；其他疾病如急性心肌梗死、心肌炎、心力衰竭所致肝脏淤血，<font color="#d99694">药物影响</font>（吃药对肝脏有损伤）等。
<font color=#07f2d7>2. 谷草转氨酶（AST）</font>
谷草转氨酶正常<40 u/L，AST 的测定可以反映<font color="#d99694">肝细胞损伤程度</font>。增高见于心肌梗死、染性肝炎、中毒性肝炎、肝癌、肝硬化活动期、脂肪肝、梗阻性黄疽胆管炎、胆囊炎等；其他疾病进行性肌营养不良、皮肌炎、肺栓塞、肾炎、胸膜炎、急性胰腺炎、溶血性疾病、药物影响等。
<font color=#07f2d7>3.总蛋白</font>
总蛋白（TP）：成人 60～80g/L，血清总蛋白<font color="#d99694">增高</font>见于 1）各种原因<font color="#d99694">脱水</font>所致的血液浓缩如呕吐、腹泻、休克、高热、肾上腺皮质功能减退等；2）血清蛋白合成增加：如多发性骨髓瘤等；血清总蛋白降低，见于各种原因引起的血清蛋白质丢失和摄入不足，如营养不良，消化吸收不良；
<font color=#07f2d7>4. 白蛋白</font>
白蛋白：成人 35~55g/L。<font color="#d99694">增高</font>见于<font color="#d99694">严重失水</font>而致的血浆浓缩。白蛋白浓度<font color="#d99694">降低</font>：1）营养不良：<font color="#d99694">摄入不足</font>、消化吸收不良。2）<font color="#d99694">消耗增加</font>：多种慢性疾病，如结核、恶性肿瘤、甲亢；或蛋白丢失过多，如急性大出血、严重烧伤、慢性肾脏病变。3）合成障碍：主要是<font color="#d99694">肝功能障碍</font> （蛋白质在肝脏合成）。

#### 肾功能检查项目
<font color=#07f2d7>1. 血肌酐（Cr）</font>
是常用的了解肾功能的主要方法之一。正常值男性 53 ~106umol/L ; 女性 44 ~97μumol/L。血肌酐增高常见于肾脏疾病、急慢性肾小球肾炎（血肌酐过滤不出来）、糖尿病、肢端肥大症、巨人症、感染、甲状腺功能减低进食肉类运动、摄入药物（如维生素 C 左旋多巴、日甲基多巴等）。
<font color=#07f2d7>2.血尿素氮（BUN）</font>
是检查肾功能的主要指标之一。正常值为 3.2-7.1mmol/ L (9-20mg / d1)。各种肾实质性病变，如肾小球肾炎、间质性肾炎、急慢性肾功能衰竭、肾内占位性和破坏性病变均可使血尿素氮增高。
<span style="background:rgba(240, 107, 5, 0.2)">以上两项都是和各种肾炎有关</span>。
<font color=#07f2d7>3. 尿酸</font>
<font color="#d99694">尿酸是人体嘌呤代谢产物</font>，我们常规体检的项目之一，是检查痛风的一种主要方法。正常成人男性为 149~417 μmol/L、女性 89~357 umol/L ；增加: <span style="background:rgba(240, 107, 5, 0.2)">痛风</span>、急慢性白血病、多发性骨髓瘤、恶性贫血、肾衰、肝衰、红细胞增多症、剧烈活动及高脂肪餐后等。如果体检检出偏高要引起重视，日常生活应控制肉食的用量，<span style="background:rgba(240, 107, 5, 0.2)">少吃红肉、动物内脏、海鲜、油炸食品, 少饮酒, 多喝白开水。粗粮、香菇、肉汤嘌呤高。</span>
##### 尿常规
尿常规是肾病检查中最常见的检查项目之一，是诊断肾脏疾病的重要手段。尿常规检查项目主要包括透明度、酸碱度、乡红细胞、白细胞、蛋白质、比重、尿胆原、胆红素、隐血、酮同体及尿糖亚硝酸盐。
在临床上尿中有重要意义的细胞为<font color="#d99694">红细胞、白细胞。</font>
<font color=#07f2d7>
①红细胞</font>：<font color="#d99694">正常人尿中可偶见红细胞</font>，镜检阴性或不超过 2 个/HP（小于三个红细胞，多于 3 个则为潜血阳性）。若尿中出现多量红细胞, 则可能由于泌尿系统结石、感染、肿瘤、急慢性肾炎、血小板减少性紫癫、血友病等
<font color=#07f2d7>②白细胞</font>：<font color="#d99694">正常可有少数白细胞存在</font>，镜检每高倍镜视野不超过 5 个。增高：见于急性肾炎肾孟盂肾炎、膀肮炎、尿道炎、尿道结核、泌尿系感染等。 
	<font color=#07f2d7>尿隐血（红细胞多于 3 个）</font>：正常参考值：阴性。临床意义：阳性，多见于炎症（肾小球肾炎、肾孟肾炎、膀胱炎等），结石（肾脏输尿管、膀胱）肿瘤（肾、膀胱输尿管）、此外，女性会偶尔出现尿潜血 （月经），剧烈运动以及服用药物等。
<font color=#07f2d7>尿蛋白</font>：正常尿液中尿蛋白阴性，如检查为阳性，见于各种急慢性肾小球肾炎、急性肾孟肾炎、多发性骨髓瘤肾移植术后等。此外，药物中毒引起肾小管上皮细胞损伤也可见阳性（<font color="#d99694">即各种炎</font>）。
<font color=#07f2d7>尿糖：</font>正常尿液中尿糖为阴性，若尿糖阳性，见于糖尿病甲状腺机能亢进、胰腺炎、胰腺癌、严重肾功能不全等。此外，颅脑外伤、脑血管意外、急性心肌梗塞等，也可出现应激性糖尿过多食入高糖食物后，也可产生过性血糖升高，使尿糖阳性。<span style="background:rgba(240, 107, 5, 0.2)">即肾功能不全或糖尿病并发症</span>。
<font color=#07f2d7>胆红素</font>：正常参考值：阴性。阳性多见于胆石症、胆道肿瘤、胆道蛔虫、胰头癌等引起的梗阻性黄疽和肝癌、肝硬化、急慢性肝炎、肝细胞坏死等导致的肝细胞性黄疽。尿胆原正常弱阳性，阳性，见于溶血性黄疽肝病等。阴性，见于梗阻性黄疽。
<span style="background:rgba(240, 107, 5, 0.2)">即肝、胆疾病</span>

#### 血脂四项
<font color=#07f2d7>1. 总胆固醇（TC）</font>: 正常值<5.20mmol/L (200mg/dl), 当血液中胆固醇过高时，就容易沉积在动脉壁上，造成<font color="#d99694">血管内皮细胞损伤</font>，导致<font color="#d99694">动脉粥样硬化</font>。
![Pasted image 20230714123701.png](/img/user/Notes/%E8%90%A5%E5%85%BB%E5%AD%A6/attachments/Pasted%20image%2020230714123701.png)
<font color=#07f2d7>2、甘油三酯（TG）</font> : 正常值<1.7mmol/L (150mg/dl)。含量过高可<font color="#d99694">使凝血倾向增高</font>，<font color="#d99694">血栓</font>形成可能性增大，促使动脉硬化的形成与发展，还可导致脂肪肝及肥胖症。
<font color=#07f2d7>3、高密度脂蛋白胆固醇</font>（HDL 一 C）：正常值>0.91mmol/L (35mg/dl)，对低密度脂蛋白有分解代谢作用，并可将周围组织细胞中的胆固醇运输到肝脏，再由肝细胞将胆固醇转化为胆酸排除体外。所以，常被称为"好"胆固醇。
<font color=#07f2d7>4、低密度脂蛋白胆固醇</font>（LDL 一 C）：正常值<3.12mmol/L (120mg/dl) : 其过高是引起动脉硬化、冠心病的重要因素（低密度脂蛋白将与胆固醇复合后，将胆固醇带到血管中）。所以，常被称为“坏”胆固醇。
- 总胆固醇高者，可能是摄入能量太多 （一般是主食吃太多），<span style="background:rgba(240, 107, 5, 0.2)">建议多餐、少食</span>；减少胆固醇摄入，如鸡蛋不要吃太多（一个鸡蛋里的胆固醇就已经是人体一天能自然代谢的胆固醇的量了）。 
- 甘油三酯高者，可能是能量摄入太多，或饮酒太多，或吃含糖食物太多。
- <span style="background:rgba(240, 107, 5, 0.2)">运动</span>可以增高高密度脂蛋白胆固醇、降低低密度脂蛋白胆固醇。

### 体检注意事项
体检报告中提到的“复查”、等术语，一定要认真对待。<span style="background:rgba(240, 107, 5, 0.2)">"复查"</span>通常是某一项检查指标此次出现异常，<font color="#d99694">可能与检查前饮食、下夜班等因素有关</font>，<span style="background:rgba(240, 107, 5, 0.2)">应在一周内进行第二次检查</span>。<span style="background:rgba(240, 107, 5, 0.2)">定期复查"</span>是指体检的结果<font color="#d99694">已有结论</font>，为了观察其变化，如在体检中发现胆囊息肉、肝 (肾) 囊肿等，<span style="background:rgba(240, 107, 5, 0.2)">需要 3 个月至半年检查一次</span>，观察大小、形状是否发生改变。
<span style="background:rgba(240, 107, 5, 0.2)">"进一步检查"</span><font color="#d99694">是指在体检中发现问题又不能确诊</font>，这样就会建议你进一步检查，需到医院专科，主要是明确诊断。特殊异常结果在最短时间内通知，需要尽快就诊治疗此时，千万不要耽误时间，应立即到医院就医。随诊即随时就诊。

## 第二节 食物的消化与吸收
<mark style="background: #F470708A;">消化</mark>：食物在消化道内被分解为小分子物质的过程。
<font color=#07f2d7>机械性消化</font>：通过<span style="background:rgba(240, 107, 5, 0.2)">消化道的运动</span>，将食物磨碎，并使其与消化液充分混合，同时将其向消化道远端推送。
<font color=#07f2d7>化学性消化</font>：通过<span style="background:rgba(240, 107, 5, 0.2)">消化液</span>的各种化学作用，将食物中的营养成分分解成小分子物质。
<mark style="background: #F470708A;">吸收</mark>：食物经过消化后，透过消化道粘膜进入<font color="#d99694">血液</font>和<font color="#d99694">淋巴循环</font>的过程。

### （一）口腔内的消化
1、睡液的性质和成分：近于中性（pH6.6～7.1）水分。
<mark style="background: #B554C861;">有机物</mark>：<font color="#d99694">粘蛋白</font>（因此唾液较黏稠）、球蛋白、唾液淀粉酶、<font color="#d99694">溶菌酶</font>（可杀菌）。
<mark style="background: #B554C861;">无机物</mark>：Na+、K +、HCO3<sup>-</sup>、CI<sup>-</sup>
2、睡液的作用 : 湿润和溶解食物；<font color="#d99694">清洁和保护</font>（<font color="#d99694">溶菌酶</font>）；使食物<font color="#d99694">合成食团</font>便于吞咽杀灭细菌和病毒（<font color="#d99694">黏蛋白</font>）；<font color="#d99694">把淀粉分解成麦芽糖</font>（<font color="#d99694">唾液淀粉酶</font>）。

### （二） 胃内的消化和吸收
#### 1 、胃液的分类
<font color=#d89eff>幽门部的胃腺</font>：分泌<mark style="background: #B554C861;">碱性粘液</mark>.
<font color=#d89eff>胃底</font>和<font color=#d89eff>胃体部</font>的胃腺
- 主细胞分泌胃蛋白酶原
- 壁细胞分泌盐酸和内因子
- 颈粘液细胞分泌粘液（含碳酸氢根离子，可中和盐酸，避免胃粘膜受到侵蚀）
#### 2 、胃液的性状
无色透明的酸性液体，pH 值 0.9～1.5，成人每日分泌 1.5 ~ 2.5L
#### 3、胃液的作用
<font color=#07f2d7>盐酸</font>
（1）激活胃蛋白酶原，为胃蛋白酶提供<font color="#d99694">酸性环境</font>
（2）<font color="#d99694">抑菌杀菌</font>
（3）进入小肠促进胰液、胆汁和小肠液的分泌
（4）有助于<font color="#d99694">小肠对铁和钙的吸收（胃酸酸性强，进入小肠后仍有酸性）</font>
<font color=#07f2d7>胃蛋白酶</font>：能将蛋白质水解为多肽和氨基酸
<font color=#07f2d7>内因子</font>：保护维生素 B12 不被小肠水解酶破坏
<font color=#07f2d7>粘液</font>：润滑、保护，与碳酸氢盐构成粘液-碳酸氢盐屏障

### （三）小肠内的吸收
#### 1、小肠内的消化液
1. 胰液 （由胰脏分泌，是最重要的消化液）
2. 胆汁  （由肝脏分泌，可消化脂肪）
3. 小肠液
##### 1. 胰液
胰液的分泌：<font color="#d99694">pH 值 7.8~8.4</font>，成人每日分泌 1～2L 
胰液的成份
	<font color=#07f2d7>无机成分</font>
	碳酸氢盐（导致胰液呈、中和胃酸）
	胰淀粉酶（<font color=#adffbb>水解淀粉</font>）
<font color=#07f2d7>有机成分</font>
	胰脂肪酶（<font color=#adffbb>分解甘油三酯</font>）
	胰蛋白酶原（<font color=#adffbb>分解蛋白质</font>）
	糜蛋白酶原（<font color=#adffbb>分解蛋白质</font>）
##### 2. 胆汁
胆汁的分泌：有苦味的有色液汁、成人每日分泌量为 800~1000mL
<font color=#07f2d7>胆汁的成分</font>：水、胆色素、胆盐、胆固醇、卵磷脂、脂肪酸、无机盐
<font color=#07f2d7>胆汁的功能</font>
	对<font color="#d99694">脂肪的消化和吸收</font>有重要作用
	胆盐<font color="#d99694">使脂肪乳化</font>成许多微滴（<font color="#d99694">增加胰脂肪酶的作用面积</font>），有利于脂肪的消化。
	胆盐可与脂肪酸甘油一酯结合，形成水溶性复合物。有利于脂肪的吸收。
##### 3. 小肠液
PH 值 7.6，成人每日分泌 1～3L
小肠液的成分：水、电解质、粘液、免疫球蛋白、肠激酶、小肠淀粉酶
小肠液的功能：消化食物和保护肠粘膜

### 四、大肠内的消化和吸收
大肠<font color="#d99694">没有重要的消化功能</font>，但有吸收功能
1）<font color="#d99694">吸收水</font><font color="#d99694">和电解质</font>
2）吸收维生素 B 和 K
3）加工食物残渣，形成并暂时储存粪便

## 第三节 不同人群的生理特点
### 一、妊娠期生理特点
妊娠期指女性从受孕到胎儿分娩，约 280 天的生理过程。

#### 1、内分泌改变
(1) <mark style="background: #F470708A;">人绒毛膜促性腺激素</mark>（HCG）在<font color="#d99694">妊娠 1 周开始出现</font>至第 9 周达高峰。此激素由胎盘分泌，<font color="#d99694">有安胎作用</font>。可以从血液、尿液中测出，<font color="#d99694">因此抽血（怀孕一周后）、试纸验尿（怀孕半个月后）可以查出是否怀孕。</font>
(2）<mark style="background: #F470708A;">人绒毛膜生长素</mark>（HCS）在妊娠 6 周出现，36 周达高峰至分娩。此激素由胎盘分泌，功能为降低母体葡萄糖利用；促进脂肪分解；促进蛋白质和 DNA 合成。
两者皆能促进胎儿发育。
(3) <mark style="background: #F470708A;">雌激素分泌增加</mark>。在妊娠 3 个月，胎盘合成大量雌激素（雌酮、雌二醇、雌三醇），可<font color="#d99694">增加子宫和胎盘的血流量、促进母体乳房发育</font>。
(4) <mark style="background: #F470708A;">孕酮分泌增加</mark>。在妊娠 8～10 周由胎盘分泌，并持续增加。生理功能为<font color="#d99694">促进子宫内膜（胎儿发育环境）和乳腺的发育</font>（因此孕酮低的容易发生流产，因子宫内膜发育不良）。
(5）在上述激素的作用下，孕妇体内的<font color="#d99694">合成代谢增强，分解代谢也增强</font>，但总体上是合成代谢>分解代谢 0。

#### 2、血容量和血液动力学变化
<font color=#07f2d7>（1）血容量增加</font>，比妊娠前约增加 35%~40%。其中，血浆容积增加 45%~50%，而红细胞数量增加 15%~20%（<font color="#d99694">因此单位体积血液中的红细胞数下降</font>）。 
<font color=#07f2d7>(2) 血液成分改变</font>
	1. <font color=#adffbb>红细胞计数下降</font>，达 3.6x10<sup>12</sup>/L （正常 4.2x10<sup>12</sup>/L）
	2. 血红蛋白值下降，达 110g/L（正常 130g/L），因此出现生理性贫血。
	3. 白细胞增加，主要为中性粒细胞
	4. 血浆总蛋白下降，达 60g/L（正常 70g/L），主要为白蛋白由 40g/L 降至 25g/L 
	5.<font color=#adffbb> 血浆所有营养素降低</font>
<font color=#07f2d7>（3）肾功能改变</font>
	1. 肾小球滤过率增加约 50%，肾血流量增加约 75%。<font color=#adffbb>肾脏负担加重</font>。
	2. 蛋白质代谢产物（尿素、尿酸、肌苷、肌酸）等排泄增多。
	3. 一些<font color=#adffbb>营养素</font>（葡萄糖、氨基酸、水溶性维生素）<font color=#adffbb>从尿中排出增加。</font>
<font color=#07f2d7>（4）消化功能的变化</font>
	1. <font color=#adffbb>牙龈肥厚（由于血液量增加）</font>（牙龈炎和出血）
	2. 平滑肌张力下降，括约肌松弛，消化液分泌下降，胃排空↑，蠕动减慢}----易出现妊娠反应<font color=#adffbb>（恶心、消化不良、呕吐、反酸、便秘等）</font>
	3.<font color=#adffbb> 但对钙、铁、维生素 B<sub>12</sub> 及叶酸等的吸收能力提升</font>（因蠕动减慢，食物在肠道中停留时间长，因此吸收效果提升）

## 一、女性生理特点
（一）女性一生的生理阶段及特点 
### 1. 新生儿期
（1）妊娠期母体及胎盘女性激素的影响，外阴较丰满，乳房略隆起或少许溢乳
（2）出生后女性激素水平迅速下降，出现少量阴道流血（因为母体怀孕时女性激素水平较高，导致婴儿女性激素水平也较高，在脱离母体后，婴儿女性激素水平降低，子宫内膜脱落，出现流血）
生理现象：<font color="#d99694">短期内的少量溢乳和阴道流血</font>
### 2. 儿童期
（1）<font color="#d99694">体格快速生长发育，生殖器发育缓慢</font>
（2）生殖器呈幼椎型，宫颈占子宫的 2/3
（3）8 岁前，下丘脑一垂体一卵巢轴的功能处于抑制状态，卵泡自主生长；8 岁后抑制解除，促性腺激素分泌，卵泡发育并分泌激素。
![Pasted image 20230727201322.png](/img/user/Notes/%E8%90%A5%E5%85%BB%E5%AD%A6/attachments/Pasted%20image%2020230727201322.png)
### 3. 青春期
（1）第一性征：生殖器官发育为成人型
（2）第二性征：体型、乳房、阴毛、腋毛
（3）生长加速：雌激素、生长激素等有关
（4）月经初潮：青春期的重要标志；卵巢功能欠成熟
### 4. 性成熟期
18 岁始，约 30 年，又称生育期
（1）卵巢周期性排卵
（2）子宫内膜周期性脱落
### 5 、更年期（ 围绝经期）
<font color=#adffbb>卵巢功能开始衰退</font>至最后一次月经的时期。
常始于 40 岁以后，短则 1-2 年，长则 10 余年。
<font color=#adffbb>雌二醇下降并波动</font>，出现血管舒缩障碍和神经精神症状，表现为<span style="background:rgba(240, 107, 5, 0.2)">潮热、出汗、情绪不稳定、失眠</span>等。
### 6、绝经后期
女性 60 岁以后，整个机体发生衰老，生殖器官萎缩、骨质疏松、易发生骨折。

## 二、女性常见健康问题的营养保健养生
### （一）美容与营养健康
什么是美?
美是一种时尚、美是一种心态、美是一种健康、美是一种自信
![Pasted image 20230730233543.png](/img/user/Notes/%E8%90%A5%E5%85%BB%E5%AD%A6/attachments/Pasted%20image%2020230730233543.png)
#### 1. 营养素与美容 
##### (1 ) 水
- 水占机体的比重大约 2/3；新生儿 3/4
- 肌肉与脂肪、皮肤含水量占人体总水量的 18-20%
- 皮肤被誉为人体水库”，只有当人体拥有了充足的水份，皮肤才会显得嫩滑、细腻、丰满，紧张而富有弹性。
![Snipaste_2023-07-30_23-43-13.png](/img/user/Notes/%E8%90%A5%E5%85%BB%E5%AD%A6/attachments/Snipaste_2023-07-30_23-43-13.png) 
##### （2）蛋白质
- 构成肌肤组织最重要的基本物质
- 是构成各种酶类、抗体和某些激素的主要成分
- 有促进<font color=#adffbb>肌肤生长发育，加快创伤修复</font>的作用；对细胞的形成，<font color=#adffbb>皮肤</font>的再生，<font color=#adffbb>头发</font>与<font color=#adffbb>指甲</font>的健康都非常重要。
##### （3）碳水化合物
- 是人体新陈代谢主要能源
##### （4 ) 维生素 A 
- 有维持皮肤组织细胞的生理功能，使皮肤柔软、细腻、光滑的功效
- 缺乏时，<font color=#adffbb>汗腺和皮脂腺萎缩</font>、功能减退、皮肤干燥、易生成皱纹、有鳞片样脱落（严重者会有鸡皮疙瘩状皮肤，有色素沉淀）。
##### （5）B 族维生素
- 参与人体蛋白质、脂肪和糖的代谢
- 可使皮肤娇嫩，富有光泽和弹性，减退色素，<font color=#adffbb>抑制炎症</font>
- VB<sub>1</sub>、VB<sub>2</sub> 有使皮肤光滑、展平皱纹、<font color=#adffbb>消除斑点</font>、<font color=#adffbb>调节皮脂腺分泌</font>的作用，维生素 B 族缺乏则导致皮指腺分泌异常
- VB<sub>1</sub>缺乏，皮肤干燥、易生皱纹（与<font color=#adffbb>神经调节</font>有关，吃 VB1 可治疗神经衰弱、失眠、心慌）；VB<sub>2</sub>, 缺乏可引发舌炎、唇炎、口角炎；VB<sub>6</sub>缺乏会发生脂溢性皮炎
![Pasted image 20230731000342.png](/img/user/Notes/%E8%90%A5%E5%85%BB%E5%AD%A6/attachments/Pasted%20image%2020230731000342.png)
![Pasted image 20230731000255.png](/img/user/Notes/%E8%90%A5%E5%85%BB%E5%AD%A6/attachments/Pasted%20image%2020230731000255.png)
##### （6）维生素 C 
- 又称抗坏血酸，是一种活性极强的还原物质<span style="background:rgba(240, 107, 5, 0.2)">（如将三价铁转为二价铁，利于铁的吸收，预防缺铁性贫血）</span>
- 有<font color=#adffbb>增加皮肤弹性</font>，<font color=#adffbb>增强抵抗力</font>及<font color=#adffbb>减轻皮肤的色素沉着</font>的作用
- 对<font color="#d99694">胶原组织的合成</font>有很大关系，对皮肤的柔软程度起决定作用
- VC 又是一种抗氧化剂，有<font color=#adffbb>抑制黑色素</font>的作用。
- 缺乏 VC 时，皮肤会出现血管变脆、皮下紫斑、色素沉着等。
##### (7) 维生素 E 
- 具有抗氧化作用，可促进和增强 VA 的作用，并与 VC 起到协同作用
- 增强皮肤的抗病能力，对胶原纤维和弹单性纤维有修复作用，改善皮肤弹性，<font color=#adffbb>促进血液循环</font>，<font color=#adffbb>使皮肤毛发、指甲光滑滋润</font>，舒展皱纹、消除色斑、延缓衰老
- 缺乏时会使皮肤干燥、粗糙，加速衰老
#### 2. 皮肤衰老的原因
导致皮肤老化的最重要的因素为遗传和过量的日光紫外线（夏天 9:30 - 16:00 紫外线特别强）。
空气、水源、噪声、森林等生活环境、社会环境、职业与劳动环境、医疗保健、运动、生活规律、烟酒等生活习惯和膳食营养状况都会影响皮肤的衰老进程。<font color="#d99694">风沙、寒冷和炎热的刺激会使皮肤老化</font>。有害粉尘、污浊空调气体、香烟烟雾也会加速老化。
#### 3. 皮肤衰老的表现
<font color=#adffbb>皮肤色素改变</font>（雀斑、老年斑、黄褐斑）、<font color=#adffbb>面部皱纹</font>（体位性、动力性（皱眉、提眉动作）、重力性皱纹（法令纹））
![Pasted image 20230731002213.png](/img/user/Notes/%E8%90%A5%E5%85%BB%E5%AD%A6/attachments/Pasted%20image%2020230731002213.png)
#### 4. 皮肤衰老的预防
（1）科学合理的饮食习惯和营养搭配：蛋白质、维生素、<font color=#adffbb>硫酸软骨素</font>（存在于动物软骨，有<font color=#adffbb>锁水</font>作用、<font color=#adffbb>修复人的关节软骨</font>，与<font color="#d99694">氨基葡萄糖</font>一起服用）
（2）精神愉快、胸襟豁达，是保持人体青春活力所不可缺少的条件。
（3）保证充分睡眠（大于 6 小时，少于 9 小时）
（4）保持适当的运动，增强体质（<font color="#d99694">一般做有氧运动，增肌的先做无氧（哑铃）再做有氧<font color=#ecd38e>（为什么不是反过来？）</font></font>）
（5）防止长时间的强烈阳光暴晒
（6）经常梳头（让气血到达头部）和面部按摩，可延缓皮肤衰老
#### 5. 不同肤质如何选择营养
##### （1）干性皮肤
干性皮肤的特点：<font color=#adffbb>皮脂分泌少</font>，<font color=#adffbb>皮肤干燥</font>，缺少光泽，并容易产生皱纹。容易受到外界物理性、化学性因素和紫外线与粉尘等影响，发生过敏反应。
外观上显得比较细腻，<font color=#adffbb>毛孔不明显</font>，无油腻感故给人以清洁、美观的感觉不易生座疮，且附着力强，化妆后<font color=#adffbb>不易掉妆</font>。但干性皮肤<font color=#adffbb>经不起外界刺激</font>，如风吹日晒等，受刺激后皮肤潮红，甚至灼痛。容易老化起皱纹，特别是在<font color=#adffbb>眼前、嘴角处最易生皱纹</font>。
<mark style="background: #B554C861;">干性缺水</mark>皮肤<font color=#adffbb>多见于 35 岁以后及老年人</font>，与<font color=#adffbb>汗腺功能减退、皮肤营养不良、缺乏维生素 A、饮水量不足等因素有关</font>。风吹、日晒也可引起皮肤缺水。
<mark style="background: #B554C861;">干性缺油</mark>皮肤<font color=#adffbb>多见于年轻人</font>，由于皮脂分泌量少，不能滋润皮肤；或护肤方法不当，<font color="#d99694">常用碱性大的香皂洗脸，导致皮肤缺油</font>。皮肤缺油常伴有皮肤缺水。
###### 干性皮肤营养调理
干性皮肤的人在饮食中要注意<font color=#adffbb>选择一些脂肪、维生素含量高的食物</font>，如牛奶、鸡蛋、猪肝、黄油、鱼类、香菇、南瓜及新鲜水果等。在秋冬干燥的季节，要格外注意保养，补充水分，多喝水，<font color=#adffbb>用油脂含量高的护肤霜</font>，防止皮肤干燥脱屑，延缓皮肤的衰老。
##### （2）混合性皮肤
混合性皮肤的特点：混合性皮肤兼有油性皮肤和干性皮肤的两种特点，<font color=#adffbb>在面部 T 区（额、鼻、口、下颌）呈油性，其余部位呈干性。</font>
混合性皮肤多见于 25-35 岁之间的人。我国大部分人都属于此类皮肤。
###### 混合性皮肤的护理
脸部中间的<font color=#adffbb>“T”字部位需要抑制皮脂分泌</font>，从而使皮肤清爽不泛油光；
<font color=#adffbb>两颊、颧骨等干燥部位需要补充水分</font>；
在早、日晚清洁脸部时，中间部位需要加强清洁。
##### (3 ) 油性皮肤
油性皮肤的特点：<font color=#adffbb>皮脂分泌旺盛</font>，多数人肤色偏深，毛孔粗大、皮肤油腻光亮，甚至可以出现桔皮样外观，<font color=#adffbb>很容易粘附灰尘和污物</font>，引起皮肤的感染与痤疮等。常见于青春发育期的年轻人。
###### 油性皮肤营养调理
饮食上最好多选用具有凉性、平性的食物，如冬瓜丝瓜、白萝卜、胡萝卜、竹笋、白菜、莲藕、西瓜、银鱼、鸡肉、兔肉等。
少吃辛辣、温热及油脂多的食物，如奶油、奶酪、蜜饯、猪肉、羊肉、狗肉、花生、桂圆肉、荔枝、核桃仁、巧克力、可可、咖喱等。
炎热的夏季，油性皮肤的人还可选用一些具有祛湿清热功效的白菊花泡水喝。

## （三）乳腺健康
### 1. 乳腺常见疾病
#### （1）经前期乳腺疼痛
- 大多数女性在月经前 1 周左右，乳房开始肿胀、疼痛，但在月经过后疼痛会自然消失。这种乳房疼痛和月经有关的情况多为正常现象。
- 乳房的疼痛与月经周期没有关系，不来月经时也会疼痛甚至在触摸乳房时能摸到肿块。如果发生这样的情况要及时去医院检查。
#### （2）乳腺增生
乳腺上皮和纤维组织增生，乳腺组织导管和乳小叶在结构上的退行性病变及进行性结缔组织的生长。其发病率占乳腺疾病的首位。

#### （4）乳腺癌
- 早期最常见的症状是乳房肿块。随着疾病的发展，会出现乳头流水（血样、水样液体），皮肤凹陷（有时皮肤会出现像橘皮一样的变化），乳头下陷等症状。
- 35～55 岁之间的女性。30 岁开始随着年龄的增长，乳腺癌发病率逐渐上升，高发年龄在 40～50 岁，绝经后 5～10 年还会出现一个乳腺癌发病的小高峰。
<font color=#07f2d7>乳腺癌的危险因素</font>
1. 乳腺癌家族遗传史
2. 乳腺密度高
3. 患乳腺良性疾病（如乳腺纤维瘤）未能及时治疗
4. 胸部因某种原因接受过高剂量放射线的照射
5. 肥胖或过多摄入脂肪
6. 精神抑郁，经常生气，心情不好
7. 未哺乳或哺乳时间过短（少于 6 个月）
8. 大于 35 岁独身未孕或婚后不孕
9. 月经初潮于 12 岁前或绝经晚
每 1～2 年应进行一次乳腺专科检查，35 岁以上的妇女应每年进行一次乳腺体检，有高危因素的人群应每半年检查一次乳腺。越早发现乳腺癌，治愈的可能性就越大，到晚期只有一部分患者有治愈的可能。因此，早发现、早诊断、早治疗尤其重要。
### 2. 乳腺病与饮食营养
（1） 高脂饮食
	高脂肪摄入增加乳腺癌的危险性，尤其是绝经后妇女饱和脂肪摄入与乳腺癌危险性存在有统计学意义的正相关。
（2）高糖饮食
	高蔗糖和高果糖饮食的小鼠的肺部肿瘤转移灶显者增多，这也提示了高糖饮食能够加速乳腺癌的转移。
（3） 维生素 A
	维生素 A 摄取不足，增加患乳腺癌的可能性。
	维生素 A 含量高的食物：动物肝脏、蛋黄、枸杞子、西兰花、胡萝卜、菠菜、芹菜叶、茴香等
(4) 维生素 D 
	获得的维生素 D 量愈高，乳腺癌死亡率就愈低维生素 D 主要来源于饮食和日照。维生素 D 含量高的食物有牛奶和鱼类。
（5）海藻类食物
	常食海带、紫菜等海藻类食物可防止乳腺癌的发生。<font color="#d99694">这类含碘、钙较高的食物能调节和平衡血液的酸碱度</font>，因而起到某种相应的防癌作用。
（6）豆类食物
	豆类食品含有大量的植物激素，经常喝豆浆，可以通过植物激素<font color=#adffbb>使女性体内的雌激素和孕激素的分泌变化趋于合理</font>（雌激素高可抑制孕激素，反之）（双向调节），降低过高的雌 (孕) 激素水平。妇女经常喝豆浆可起到调整内分泌的作用，从而起到预防乳腺癌的作用。

# 慢病营养指导
## 第一节 肥胖
### 一、肥胖的定义
- 肥胖（obesity）是<font color="#d99694">能量摄入超过能量消耗而导致体内脂肪积聚过多达到危害程度</font>的一种慢性代谢性疾病。
- 肥胖及其相关慢性病已成为威胁人类健康的重要公共卫生问题。
### 二、肥胖的分类
肥胖<font color="#d99694">按发生原因</font>可分为三类
①遗传性肥胖：主要由遗传物质（染色体、DNA）发生改变而引起，比较罕见；
②继发性肥胖：主要由于下丘脑-垂体-肾上腺轴发生病变内分泌紊乱或其他疾病、外伤引起的<font color="#d99694">内分泌障碍</font>导致；
③单纯性肥胖：单纯由于<font color="#d99694">营养过剩</font>所造成的全身性脂肪过量积累，也常表现为家族聚集倾向。
### 三、肥胖的诊断方法及危害
#### 1 . 肥胖的测量和诊断方法
常用的方法有<font color=#adffbb>人体测量法</font>、<font color=#adffbb>物理测量法</font>和<font color=#adffbb>化学测量法</font>
##### （1）人体测量法：主要包括身高标准体重法、皮褶厚度和体质指数
###### 1 ）身高标准体重法
- WHO 推荐法
$男性：[身高\,(cm)－80]×70﹪=标准体重$
$女性：[身高 (cm)－70]×60﹪=标准体重$
标准体重正负 10﹪为正常体重
- 不同年龄算法
	标准体重（kg） = 年龄 x 8 + 8 （ 7 - 16 岁 ）
$$肥胖度(\%) =\frac {实际体重(kg)- 身高标准体重(kg)} {身高标准体重重 (kg )} * 100\% $$
<mark style="background: #F470708A;">判断标准</mark>
- 肥胖度≥10%为超重; 
- 20%~29%为轻度肥胖；
- 30%~49%为中度肥胖；
- ≥50%为重度肥胖；
- >100%为病态肥
标准体重计算有不同方法，这仅作参考

###### 3）皮褶厚度法
- 主要是用<font color="#d99694">皮褶厚度测量仪</font>测量<font color=#adffbb>肩下</font>和<font color=#adffbb>上臂肱三头肌腹处</font>皮褶厚度，二者加在一起即为皮褶厚度。
还可测量<font color=#adffbb>骼骨上和脐旁 1cm 处</font>皮褶厚度。皮褶身厚度一般不单独作为判定肥胖标准，一般和体重结合起来判定。

####     2. 肥胖对健康的危害
肥胖对人体健康的危害包括代谢综合征的危险性增高，易发生糖尿病、心血管疾病、意外伤害、抑郁、血栓性疾病等。对于儿童，肥胖不但可影响儿童的心血管系统、呼吸系统、内分泌系统与免疫系统，还对儿童的生长发育、智力、心理行为有不良影响。

### 四、营养与肥胖的关系
肥胖发生的根本原因是机体能量摄入大于机体的能量消耗，从而使多余的能量以脂肪形式储存，最终导致肥胖。肥胖发生的根本原因从大体可以分为内因和外因。内因主要是指肥胖发生的遗传生物学基础，外因主要包括饮食营养因素和体力活动，环境因素（即外因）在肥胖的发病中占 30%~60%，或甚至更高。
#### （1）饮食、营养因素
##### ①生命早期营养状况
此期的机体处于旺盛的细胞分裂增殖、分化和组织器官形成阶段，对外界各种刺激非常敏感，并且会产生记忆（又称代谢程序化）持续到成年，对<font color=#adffbb>成年后的肥胖</font>及相关慢性病的发生、发展有重要影响。目前比较公认的是，<font color=#adffbb>妊娠期孕妇营养缺乏或过剩</font>，完全人工喂养、过早断乳、过早添加辅食或婴幼儿期营养过剩，不仅可直接影响婴幼儿体重及健康，而且还会<font color=#adffbb>增加成年后肥胖及相关慢性病的发病风险性。</font>
##### ②膳食结构不合理
合理的膳食结构是指根据膳食参考摄入量而确定的食物摄入种类、数量和比例。膳食结构不合理主要表现为<span style="background:rgba(240, 107, 5, 0.2)">成人谷类和根茎类食物消费下降</span>；<span style="background:rgba(240, 107, 5, 0.2)">动物性食物</span>尤其是畜类和<font color=#d89eff>蛋类</font>食物呈<span style="background:rgba(240, 107, 5, 0.2)">明显上升趋势;</span> 另外油<span style="background:rgba(240, 107, 5, 0.2)">脂类消费亦呈明显增加趋势</span>。高脂肪膳食可增加肥胖的危险性或诱导肥胖发生。
##### ③摄食量过大
能量摄入过多，相关的因素有遗传因素，社会环境及心理因素，个人饮食习惯等。
#### （2）体力活动因素
- 包括<font color=#adffbb>步行时间明显减少</font>、体力劳动强度明显减轻、体力活动明显减少
- 体力活动减少，影响了能量平衡的调节，导致能量消费减少，能量过剩，是导致肥胖发生的一个非常重要的因素。（<font color=#d89eff>因此只节食、不运动往往减肥效果不好</font>）
### 五、肥胖的营养防治
肥胖的防治不仅要关注体重、体脂的减少，还要重视<font color=#adffbb>代谢紊乱</font>的纠正。肥胖的膳食治疗原则是<font color=#adffbb>达到能量负平衡促进脂肪分解</font>。自前常采用的方法有以下几点：
#### 1、措施
##### （1）控制总能量摄入
这是首要措施限制<font color=#adffbb>每天的食物摄入量</font>和摄入食物的<font color=#adffbb>种类</font>，以便减少摄入的热能。减少热能必须以保证人体能从事正常的活动为原则。
<mark style="background: #FFD25C7A;">平时</mark>推荐的膳食构成蛋白质、脂肪和碳水化合物的供能比分别为 15%、< 30%和 50% ~ 55% ; 建议<font color=#adffbb>每天减少 500kcal 的能量摄入</font>，能量减少过多，不仅会影响或损害身体健康，而且难以坚持，依从性差。
##### （2）改变宏量营养素构成
常用的减肥膳食包括低脂、低碳水化合物和高蛋白质膳食该膳食不仅可<font color=#adffbb>有效减轻体重，改善代谢紊乱</font>，而且可<font color=#adffbb>增加饱腹感</font>，提高依从性，有利于减肥后体重的维持，<font color=#adffbb>防止反弹</font>。低碳水化合物膳食在某些方面优于低脂膳食，但该结论目前尚有争议。
比较公认的是，<mark style="background: #B554C861;">高蛋白（供能比占 20%~25%）低脂肪（供能比占 30%以下）、低碳水化合物（供给比占 45%~50%）</mark>膳食的减体重<font color=#adffbb>效果最佳</font>。
#### （3）保证维生素和矿物质的供给
应保证充足的维生素、矿物质和微量元素的供应。<font color="#d99694">新鲜蔬菜、水果、豆类、动物内脏如肝脏、牛奶</font>等是维生素和矿物质的主要来源。另外，在医生的指导下，可以适当服用多种维生素和无机盐制剂。
#### （4）增加膳食纤维的供给
含<font color=#adffbb>膳食纤维高</font>的食物（粗粮、正蔬菜、水果等）对健康非常有利，尤其对肥胖者。凡膳食纤维多的食物均可适当多用，每人每天膳食纤维供给量以<font color=#adffbb>不低于 12g </font>为宜。
#### （5）补充多不饱和脂肪（n-3 脂肪酸）
单不饱和脂肪酸等营养素和异黄酮、皂苷等植物化学物不仅有助于减肥，而且还能改善代谢紊乱。

#### （6）运动
任何肥胖的膳食治疗方案，都应配合运动，以便取得更好的减肥效果因为运动有助于维持减肥状态，防止反弹；改善代谢紊乱；改善心情和健康状态。此外还可预防多种慢性疾病，如心脏疾病、糖尿病、癌症等，甚至降低死亡率。通常的做法是，运动疗法和节食法并用会取得更有效的减肥效果。

>[!tip]-  # 各种茶的功效
>## 綠茶：含兒茶素最高
>兒茶素具有高度抗氧化性。兒茶素可抑制腸道內澱粉分解酵素的活性，降低腸道吸收葡萄糖的速度，因此能**減緩飯後血糖上升**。也有研究指出，兒茶素可抑制脂肪組織增生、促進細胞新陳代謝，**降低脂肪堆積**。
>此外，**運動時喝茶更有助於燃脂**，由於運動時會刺激人體正腎上腺素分泌，誘發脂解酶作用，將脂肪轉化成運動時所需能量，本就有消脂作用，不過，人體中另一種酵素 COMT 會分解正腎上腺素，影響脂肪代謝，如果同時喝茶，當中的兒茶素恰好能抑制 COMT 作用，便能提高 17％的燃脂效率。
>## 烏龍茶：飢餓素促進腸胃蠕動
然而，除了兒茶素之外，2011年刊登在《營養》期刊中的研究發現，其他茶多酚如茶黃素、茶紅素也有消脂效果，而烏龍茶、紅茶中都存在茶黃素、茶紅素。人體研究發現，連續喝6周的烏龍茶，可降低血脂肪濃度。中興大學生物科技學研究所教授曾志正實驗發現，烏龍茶含有**飢餓素**，可促進腸胃排空、刺激生長激素分泌等功效，但同時也會**有飢餓感**。
烏龍茶中的飢餓素又稱「茶飢素」，可提高 5 倍食欲，使喝茶者感覺飢餓，因此促進腸胃蠕動。不過有趣的是，在茶飢素的作用下，身體運作活化了，肚子雖餓，卻又不會提高食量，因此不怕吃太多而變胖。
>## 普洱茶：單寧酸阻止小腸吸收脂肪
普洱茶有**抑制肝臟膽固醇合成**的效果，並可有效**降低血漿中的膽固醇、三酸甘油酯和游離脂肪酸**，增加糞便中的膽固醇排除量，同時抑制低密度脂蛋白（LDL）氧化。
而普洱茶含有的水解型鞣酸單寧，能有效抑制胰脂解酵素產生，因此我們**吃下肚的油脂無法被小腸吸收**，就會**跟著糞便排出體內**。「很多人喝了普洱茶後，糞便變軟，正是因為裡面的油脂含量高。」利用吃多但身體不吸收的方式，自然能有維持體重，甚至減肥的效果。

##### （1）运动本身消耗能量
尽管高强度的运动能量消耗较低，但是<font color=#adffbb>高强度运动皮褶减少程度比低强度运动高 9 倍</font>。那些维持高强度运动计划的减肥者更能维持体重，甚至使脂肪减少更为突出。<font color=#adffbb>减肥初期以中强度为主</font>，更多消耗热量；减肥后期增加高强度运动，让皮肤更紧致。
有研究发现在平均减肥 15 干克，维持时间 1 年以上的人群中，每周应消耗 2800 干卡的热量才能**保持减肥的效果**

剔除日常散步溜达，专门迈开大步走，约 1 万步。
##### （2）运动可以提高基础代谢率
人体每天能量消耗中基础代谢占 60-70%，体力活动占 5-30%，食物热效应占 10%左右。
<font color=#adffbb>单纯控制饮食</font>可以降低体重，但除了脂肪组织减少外，<font color=#adffbb>瘦体重也会丢失</font>，基础代谢率会降低；另外建立起新的能量平衡后，机体储诸存的脂肪消耗也较少，所以单纯限制饮食使体重下降到一定水平后，体重下降速度减慢或者不再下降。
>在肥胖女性人群（BMI 平均值 32kg/m2 左右）的研究中发现，运动（平均每周 280 分钟）加饮食控制（能量减少 500-1000kcal/d）在 6 个月后减少体重 12%，12 个月后减少体重 14%，到第 18 个月还能维持减肥 14%左右，而运动量较少者（每周 150-200 分钟），减肥效果较差，到第 12 个月时体重有回升现象。

大多数人在进行可持续的骑自行车，慢跑或游泳运动时，代谢速度比安静时高出 8-10 倍；**高强度的运动之后，代谢速度升高的状态可以一直持续 15 个小时**（因此运动后大吃也没大问题）。中国居民膳食指南建议超重和减肥者抗阻肌肉力量锻炼隔天进行，每次 10-20 分钟。

## 第二节营养与糖尿病
### 一、糖尿病的定义、诊断与分类
#### 定义
糖尿病是一种以慢性血葡萄糖（简称<font color=#adffbb>血糖</font>）<font color=#adffbb>水平增高</font>为特征的代谢性疾病，是由于机体<font color=#adffbb>胰岛素分泌缺陷</font>和/或<font color=#adffbb>胰岛素作用缺陷</font>引起。
糖尿病的临床表现为<font color=#adffbb>“三多一少”（多食、多饮、多尿及体重减少）</font>，久病可引起多个系统的损害，病情严重时或应激时可发生代谢紊乱和酮症酸中毒。
<font color="#d99694">胰岛素抵抗</font>（IR）是指胰岛素作用的靶器官 (（脂肪、肌肉）对胰岛素作用的敏感性下降（不听胰岛素控制），即正常剂量的胰岛素产生低于正常生物学效应的一种状态，被认为是 2 型糖尿病的发病基础。
#### 诊断
WHO 对诊断和分类统一了标准：<font color="#d99694">有糖尿病症状</font>、随机血糖≥11.1mmol/L（200mg/dl）和（或）空腹血浆葡萄糖（FPG）≥7.8mmol/L（140mg/dl）可诊断为糖尿病。
#### 分类
根据不同的病因，糖尿病可分为
①1 型糖尿病（T1DM），因胰腺β细胞破坏，导致胰岛素分泌绝对缺乏所致。
②2 型糖尿病（T2DM），可由<font color="#d99694">以 IR 为主</font><span style="background:rgba(240, 107, 5, 0.2)">伴胰岛素分泌不足</span>转为以胰岛素分泌不足为主伴 IR。
③妊娠期糖尿病（GDM），一般在妊娠后发生，占妊娠妇女的 2%～3%，大部分患者分娩后血糖可恢复正常。
④其他类型糖尿病，由某些内分泌疾病、胰腺疾病感染、药物及化学制剂引起。
### 二、糖尿病的危险因素
①遗传因素：糖尿病有家族遗传易感性
②营养因素：营养不平衡，<font color=#adffbb>长期摄入高能量、高脂肪。低膳食纤维</font>的膳食以及<font color=#adffbb>某些维生素和矿物质不足</font>，易诱发糖尿病和肥胖，超重和肥胖也是糖尿病的重要危险因素。
③缺之体力活动
④生理因素：年龄增大，妊娠等。糖尿病随年龄的增长发病率上升，<font color=#adffbb>50～70 岁是大多数糖尿病</font>的发病年龄
5 社会环境因素：不良生活方式，如吸烟、过量饮酒，生活节奏加快、竞争激烈、压力大、应急增多等。
6 病理因素：高血脂、高血压、肥胖（<font color=#adffbb>尤其是中央型，即内脏型肥胖</font>）、感染、应激、化学毒物等。
### 三、营养与糖尿病的关系
目前对于糖尿病发病的营养因素研究主要集中在营养物质代谢过程中对胰岛素分泌的影响，尤其是<font color=#adffbb>碳水化合物和脂肪的代谢</font>。
#### （1）能量代谢
糖尿病病人体内因<font color=#adffbb>胰岛素缺乏</font>或<font color=#adffbb>胰岛素受体数目减少</font>，组织对胰岛素不敏感，易发生能量代谢的紊乱。能量摄入过低易引发酮血症，摄入过高则加重病情。
#### (2）碳水化合物
糖尿病代谢紊乱的主要标志是高血糖，并可引起全身性的代谢紊乱。糖尿病患者<font color=#adffbb>过高摄入碳水化合物时</font>，机体调节血糖的功能失控极易出现<font color=#adffbb>高血糖</font>；但碳水化合物摄入不足时，体内需动员脂肪和蛋白质分解供能，引起酮血症。
食物中碳水化合物的分子量及结构不同，致餐后血<font color="#d99694">糖升高的快慢及幅度</font>也不同，其影响程度可用血糖生成指数（glycemic index，GI）来衡量。一般情况下<font color=#adffbb>血糖指数越低的食物对血糖升高的反应越小</font>。低 GI 食物可以有效控制餐后血糖，有利于血糖浓度保持稳定。
<font color="#d99694">单糖类和双糖类</font>比多糖类更易通过肠道上皮细胞进入血液，餐后<font color="#d99694">血糖的升高也较为迅速</font>，刺激胰岛素分泌的作用较强。不同结构的多糖类碳水化合物引起的血糖反应也不相同，直链淀粉不产生胰岛素抗性，支链淀粉可使血糖和胰岛素水平明显升高。
#### （3）脂类
摄入高脂膳食时，<font color=#adffbb>脂肪的氧化分解</font>消耗大于葡萄糖分解的中间产物（如α-磷酸），<font color=#adffbb>阻断了葡萄糖的彻底氧化分解</font>，使血糖浓度上升，胰岛素分泌增加；而游离脂肪酸的浓度较高，<font color=#adffbb>肌肉摄取脂肪酸进行氧化供能</font>的作用增强，从而<font color=#adffbb>使葡萄糖的利用减少</font>，出现胰岛素抵抗。膳食饱和脂肪酸、反式脂肪酸是糖尿病的危险因素，而多不饱和脂肪酸，特别是长链 n-3 系列却能改善糖代谢和胰岛素敏感性。
正常人的脂类代谢处于动态平衡状态。糖尿病患者由于糖脂代谢异常，可引起<font color=#adffbb>脂肪肝，酮血症和酮尿造成代谢性酸中毒，严重者表现为酮症酸中毒</font>、高渗性昏迷。为防止酮血症和酮症酸中毒，需要适量地供给碳水化合物，减少体脂的过多动员氧化。
#### （4）蛋白质
目前还无确切的证据表明膳食蛋白质含量与糖尿病发病有直接关系，但蛋白质代谢与碳水化合物和脂肪代谢密切相关。当碳水化合物和脂肪代谢出现紊乱时，蛋白质的代谢也必然处于不平衡状态可引起胰岛素分泌量的变化，促进糖尿病的发生。<span style="background:rgba(240, 107, 5, 0.2)">（间接影响）</span>
#### （5）矿物质和维生素
目前还没有关于矿物质和维生素对糖尿病发病的深入系统的研究，普遍认为膳食<font color=#adffbb>补充三价铬和适量补硒</font>对糖尿病有积极地预防和辅助治疗作用，三价铬是葡萄糖耐量因子的主要组成部分，也是胰岛素的辅助 r 因子可促进葡萄糖的利用，改善糖耐量。
充足的维生素对调节机体的物质代谢有重要作用，<font color=#adffbb>B 族维生素</font>（维生素 B1、维生素 B2、维生素 PP）参与<font color=#adffbb>糖类代谢</font>，<font color=#adffbb>维生素 E、维生素 C、β-胡萝卜素</font>等具有<font color=#adffbb>抗氧化作用</font>的维生素能帮助消除积聚的自由基，防止生物膜的脂质过氧化。
#### （6）膳食纤维 
膳食纤维有<font color=#adffbb>降低空腹血糖</font>和<font color=#adffbb>延缓碳水化合物吸收</font>、<font color=#adffbb>降低餐后血糖</font>及改善葡萄糖耐量的作用，是降低 2 型糖尿病高危因素的重要膳食成分，所以糖尿病患者应多摄入膳食纤维含量丰富的粗粮，建议膳食纤维的供给量为 30g/d。
### 四、膳食原则
糖尿病是一种病因尚不干分明确的慢性代谢性疾病。糖尿病的治疗应是综合治疗，主要包括以下五点：①健康教育；②营养治疗（饮食治疗）；③运动疗法；④药物治疗；5、糖尿病自我检测，其中营养治疗则是控制血糖最基本、最有效的治疗措施之一。
合理地控制饮食有利于控制糖尿病的病情发展尤其是轻型患者（空腹血糖≤11.1mmol/L）<font color="#d99694">单纯采用营养治疗</font>即可达到控制血糖的目的。
糖尿病营养治疗的原则：有效<font color="#d99694">控制每日总能</font>的摄入，三大产能营养素（蛋白质、脂肪、碳水化合物）比例适中；食物应多样化；饮食结构和餐次合理分配；注重微量养素的补充，食谱应因人而异。
#### （1）合理控制总热能，维持理想体重为宜
合理控制总能量摄入是糖尿病营养治疗的首要原则。总能量应根据患者的标准体重、生理条件、劳动强度、工作性质而定。
对于<font color="#d99694">正常体重</font>的糖尿病患者，能量摄入以<font color="#d99694">维持或略低于理想体重</font>为宜。肥胖者应减少能量摄入以控制体重，消瘦者应适当提高能量摄入以增加体重。儿童、孕妇、乳母应增加维持其特殊生理需要和生长发育的能量消耗。
#### （2）碳水化合物不宜控制过严
碳水化合物供给量以占总能量的 50%~60%为宜，一般成年患者每日碳水化合物摄入量应控制在 200～350g，折合主食为 250g ~ 400g。
除了注意碳水化合物的摄入量，还应注意食物种类、淀粉类型（升糖指数）、烹调方式对餐后血糖的影响。计算碳水化合物的量及其在食物中的供能比例时，还要考虑食物的<font color=#adffbb>血糖生成指数</font>（GI）。糖尿病患者应选择低 GI 的食物。一般来说，<font color=#adffbb>粗粮的 GI 低于细粮</font>，<font color=#adffbb>复合碳水化合物低于精制糖</font>。
故糖尿病患者宜多食用粗粮和复合碳水化合物，少用富含精制糖的甜点。为了改善食品的口味，必要时可<font color=#adffbb>选用甜叶菊、木糖醇等甜味剂</font>代替蔗糖。若食用水果，应当减少主食量。
#### （3）脂肪摄入量不宜过高
为防止或延缓糖尿病患者的心脑血管并发症，必须限制膳食脂肪的摄入量，尤其是饱和脂肪酸。脂肪摄入量占总能量较合适的比例为 20%~25%，最高不应超过 30%，其中饱和脂肪酸占总能量应少于 10%。胆固醇摄入量应少于 300mg/d，合并高脂血症者，应低于 200mg/d。因此，糖尿病患者应<font color=#adffbb>避免进食</font>富含胆固醇的食物，如<font color=#adffbb>脑、肝、肾等动物内脏及蛋黄</font>等。
#### （4）蛋白质
糖尿病患者糖异生作用增强，蛋白质消耗增加，易出现<font color="#d99694">负氮平衡（？）</font>，因此应保证蛋白质的摄入量，占总能量的 12%~20%，其中应有 1/3 以上的蛋白质为优质蛋白质。增加蛋白质摄入时，应注意观察肾功能，对合并肾病变者，增加蛋白质摄入应慎重。
#### （5）维生素和矿物质应满足需要
糖尿病患者因<font color="#d99694">主食和水果摄入量受限制</font>，<font color="#d99694">且体内物质代谢相对旺盛</font>，较<font color="#d99694">易发生维生素和矿物质缺乏</font>。调节维生素和矿物质的平衡，有利于糖尿病患者纠正代谢紊乱、防治并发症。因此，供给足够的维生素和矿物质也是糖尿病营养治疗的原则之一，其中比较重要的有<font color="#d99694">维生素 C、维生素 E、β-胡萝卜素、部分 B 族维生素</font>等。
<font color=#adffbb>锌、铬、硒、锰、锂有利于改善糖耐量，促进胰岛素的合成和分泌</font>。在保证矿物质基本供给量的基础上，还可适量增加钾、镁、钙、铬、锌等元素的供给。但<font color=#adffbb>应限制钠盐摄入</font>，以防止和降低高血压、高血脂、动脉硬化和肾功能不全等并发症的发生。
#### （6） 膳食纤维
膳食纤维具有较好的治疗糖尿病的作用，能有效的改善糖代谢，降血压，降血脂和防止便秘等。建议膳食纤维供给量 20~35g/d 或 15~25g/1000kcal。

以及糙米、大豆、豌豆等。
#### （7）少饮酒
使用人工甜味剂也不宜过多：血糖控制不佳的糖尿病患者不应饮酒。对血糖控制良好的患者可适量饮酒，但需严格设计饮食计划。总之，对平时不饮酒的患者不鼓励饮酒，对有饮酒习惯的患者在病情稳定情况下不强调戒酒，但要控制饮酒量。
#### （8）饮食分配及餐次安
根据血糖、尿糖升高时间用药时间和病情是否稳定等情况，并结合病人的饮食习惯合理分配餐次，<font color="#d99694">至少一日 3 餐</font>，定时、定量，<font color="#d99694">早、中晚餐能量各占 1/3，或 1/5、2/5、2/5 的能量比例</font>分配。如加餐，<font color=#adffbb>加餐量应从正餐的总量中扣除，做到加餐不加量</font>。在总能量范围内，适当增加餐次有利于改善糖耐量并可预防低血糖的发生。

### 六、糖尿病患者食谱
2 型糖尿病可以吃的主食
	玉米、小米、薏米、黑米、燕麦、荞麦、莜麦、红小豆、绿豆、黑豆。
2 型糖尿病患者尽量少吃的
	五谷制品面包、绿豆糕、红豆沙、年糕、粽子、油面筋、挂面、花卷、烙饼、烧饼、馒头。
2 型糖尿病患者不宜吃的五谷及其制品
	油条、月饼、麻花、饼干、蛋糕、方便面、汉堡包、三明治、比萨饼、锅巴、爆米花、西谷米。

1、糖尿病患者的主食一般以米、面为主，但是，我们<font color="#d99694">比较喜欢粗杂粮，如燕麦、麦片、玉米面</font>等，因为这些食物中<font color="#d99694">有较多的无机盐、维生素，又富含膳食纤维</font>，膳食纤维具有减低血糖作用，对控制血糖有利。
2、糖尿病患者的蛋白质来源，<font color="#d99694">大豆及其豆制品为好</font>，一方面，其所含蛋白质量多质好另一方面，其<font color="#d99694">不含胆固醇，具有降脂作用</font>，故<font color="#d99694">可代替部分动物性食品，如肉类</font>。
3、糖尿病患者在控制热量期间，<font color="#d99694">仍感饥饿时，可食用含糖少的蔬菜</font>，用<font color="#d99694">水煮后加一些佐料拌着吃</font>。由于蔬菜<font color="#d99694">所含膳食纤维多</font>、水分多，供热能低、具有饱腹作用，是糖尿病患者必不可少的食物。
4、禁用食物有：<font color="#d99694">白糖、红塘、葡萄糖及糖制甜食</font>，如糖果、糕点、果酱、蜜饯、冰激凌、甜饮料等。另外，含碳水化合物较多的土豆、山药、芋苏、藕、蒜苗、胡萝卜等少用或食用后减少相应的主食量。
5、<font color="#d99694">富含饱和脂肪酸的猪油、牛油、洋油、奶油、黄油等少用</font>，最好不用。可<font color=#adffbb>用植物油代替部分动物油</font>，<font color=#adffbb>花生、核桃、芝麻、瓜子中含脂肪也相当多</font>，尽量不吃或少吃或减少油类摄入。
（花生、核桃含的不是不饱和脂肪酸吗？不能多吃？）
6、蛋黄和动物内脏如肝、脑、腰等含胆固醇相当高，应尽量少用或不用
7、水果中含葡萄糖、果糖，能使血糖升高，故在血、尿糖控制相对稳定时, 空腹血糖<7.8mmol/ L 或餐后 2 小时血糖<10mmol/L 时，可在两餐或临睡前食用，但也要减少相应主食。
8、酒类，主要含酒精，产热高，而其他营养素含量很少，故不饮为宜
9、糖尿病患者的饮食除控制总热卡外，还应做到食品多样化，但因为限制糖、盐，使菜肴味道较单一。针对这一点，市场上生产了多种甜味剂如甜味菊、甜味糖，其不产热、不含任何营养素，近两年有一种甜味剂为蛋白糖，其有氨基酸组成，也不产热，无任何副作用，是目前较理想的甜味剂。
10、对于胰岛素依赖型的患者，同样需要在医生和营养师的指导下严格执行饮食控制, 对肥<font color="#d99694">胖合并有高血压、冠心病的糖尿病患者，除了较严格的饮食控制外，忌食动物内脏、蛋黄、鱼子等，严格控制动物油如黄油、猪油、牛油等</font>，其中的饱和脂肪酸对预防动脉粥样硬化不利
11、对于合并肾脏功能不全的糖尿病患者，除控制总热量外，应根据病情注意<font color="#d99694">少盐，无盐或少钠</font>及蛋白质的摄入量，蛋白质供应不宜过高，并且忌食豆制品，对于尿毒症应低蛋白饮食，蛋白质每天在 30 克左右，主食以麦淀粉代替米、面、蛋白质供给，<font color=#adffbb>首选优质蛋白质，如牛奶、鸡蛋、瘦肉</font>等。
